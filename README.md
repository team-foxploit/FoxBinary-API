## Instructions

- Install virtualenv
```
$ python -m pip install virtualenv
$ py -m venv env
```
- Activate the environment.
```
# MacOS:
  $ source bin/activate
# Windows:
  $ env\Scripts\activate
# Linux:
  $ env/Scripts/activate
```

- Install dependencies
```
  $ python -m pip install -r requiremenets.txt
```

- Run server
```
  $ python manage.py runserver [TODO]
  $ python server.py
```
